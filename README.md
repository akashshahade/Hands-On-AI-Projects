## Getting Started

### Prerequisites

```bash
# Required
Python 3.8+
pip or conda

# Recommended
Git
VS Code or PyCharm
OpenAI API key (for LLM projects)
```

### Installation

1. **Clone the repository**
```bash
git clone https://github.com/yourusername/Hands-On-AI-Projects.git
cd Hands-On-AI-Projects
```

2. **Choose a project**
```bash
cd generative-ai/ai-story-generator
```

3. **Install dependencies**
```bash
pip install -r requirements.txt
```

4. **Set up environment variables**
```bash
cp .env.example .env
# Add your API keys to .env
```

5. **Run the project**
```bash
streamlit run main.py
# or
python main.py
```
